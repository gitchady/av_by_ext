# Roadmap: AV.by Currency Converter Extension

Этот документ содержит подробный план разработки браузерного расширения для конвертации цен на портале av.by.

## 🎯 Основная цель
Создать легкое, безопасное и устойчивое к изменениям сайта расширение, которое отображает эквивалент стоимости автомобилей в USD/EUR/RUB рядом с основной ценой в BYN.

---

## 🛠 Технологический стек
* **Manifest V3** (современный стандарт безопасности Google).
* **Vanilla JS** (для максимальной производительности).
* **Chrome/Web Storage API** (для кэширования курсов и настроек).
* **Alarms API** (для периодического обновления данных в фоне).
* **MutationObserver** (для работы с динамическим контентом/бесконечным скроллом).

---

## 📋 Этапы разработки

### Этап 1: Архитектура и Безопасность (Policy First)
- [ ] **Manifest Setup:** Настройка `manifest.json` с минимальными `host_permissions` (`*://*.av.by/*`, `*://*.nbrb.by/*`).
- [ ] **Background Service Worker:** Создание `background.js` для работы с API Нацбанка (NBRB).
- [ ] **Privacy Policy:** Подготовка статической страницы с политикой конфиденциальности (требование Store).

### Этап 2: Слой данных (Data Layer)
- [ ] **API Integration:** Реализация запроса к `nbrb.by/api/exrates/rates/431` (USD).
- [ ] **Caching Logic:** Сохранение курса в `chrome.storage.local` с меткой времени (обновление раз в 24 часа).
- [ ] **Custom Rate:** Логика приоритета: если в настройках задан "свой курс", использовать его вместо API.

### Этап 3: Устойчивый парсинг (Anti-Fragile DOM)
*Чтобы расширение не сломалось при обновлении сайта:*
- [ ] **Hybrid Selectors:** Использование селекторов по атрибутам и регулярных выражений (Regex) для поиска текста вида `45 000 р.`.
- [ ] **MutationObserver:** Настройка слежения за контейнером объявлений для обработки новых элементов при подгрузке страницы.
- [ ] **Price Sanitization:** Функция очистки строки цены от пробелов и спецсимволов для корректных математических расчетов.

### Этап 4: Пользовательский интерфейс (UI/UX)
- [ ] **In-Page Injection:** Вставка цены в $ с копированием стилей оригинального элемента (шрифт, цвет, размер).
- [ ] **Popup Menu:** Создание окна настроек:
    - Переключатель валют (USD/EUR/RUB).
    - Переключатель источника (Авто/Ручной ввод).
    - Поле для ввода своего курса.
- [ ] **Instant Update:** Механизм `chrome.runtime.sendMessage` для мгновенного пересчета цен на странице при смене настроек.

### Этап 5: Финализация и деплой
- [ ] **Asset Preparation:** Создание иконок (16x16, 48x48, 128x128).
- [ ] **Testing:** Проверка на разных типах страниц (поиск, карточка авто, личный кабинет).
- [ ] **Store Submission:** Загрузка в Chrome Web Store / Firefox Add-ons.

---

## 🚀 Будущие улучшения (Roadmap v2)
- **Remote Config:** Загрузка CSS-селекторов с внешнего JSON-файла (GitHub Gist), чтобы фиксить баги верстки без перевыпуска расширения.
- **Custom Tooltip:** Подсказка при наведении: "Курс НБРБ на [Дата]".
- **Auto-Duty:** Калькулятор растаможки для авто из-за границы (Указ №140).
